# Day 4: 30 Days of Python Programming

- Concatenate the string `'Thirty'`, `'Days'`, `'Of'`, `'Python'` to form `'Thirty Days Of Python'`.
- Concatenate the string `'Coding'`, `'For'`, `'All'` to form `'Coding For All'`.
- Declare a variable named `company` and assign it `"Coding For All"`.
- Print the variable `company`.
- Print the length of `company` using `len()`.
- Convert all characters of `company` to uppercase using `upper()`.
- Convert all characters of `company` to lowercase using `lower()`.
- Use `capitalize()`, `title()`, and `swapcase()` on `company`.
- Slice out the first word of `company`.
- Check if `company` contains the word `'Coding'` using methods like `index()` or `find()`.
- Replace the word `'Coding'` in `company` with `'Python'`.
- Change `'Python for Everyone'` to `'Python for All'` using `replace()` or other methods.
- Split `'Coding For All'` by space.
- Split `"Facebook, Google, Microsoft, Apple, IBM, Oracle, Amazon"` by comma.
- Find the character at index `0` in `company`.
- Find the last index of the string `company`.
- Find the character at index `10` in `'Coding For All'`.
- Create an acronym for `'Python For Everyone'`.
- Create an acronym for `'Coding For All'`.
- Use `index()` to find the first occurrence of `'C'` in `company`.
- Use `index()` to find the first occurrence of `'F'` in `company`.
- Use `rfind()` to find the last occurrence of `'l'` in `'Coding For All People'`.
- Use `index()` or `find()` to find the first occurrence of `'because'` in:  
  `'You cannot end a sentence with because because because is a conjunction'`.
- Use `rindex()` to find the last occurrence of `'because'` in the above sentence.
- Slice out `'because because because'` from the above sentence.
- Check if `'Coding For All'` starts with `'Coding'`.
- Check if `'Coding For All'` ends with `'coding'`.
- Remove leading and trailing spaces from `'   Coding For All      '`.
- Check which of the following returns `True` for `isidentifier()`:

- Join the list `['Django', 'Flask', 'Bottle', 'Pyramid', 'Falcon']` with `"# "` separator.
- Use newline `\n` to separate the sentences:

  `"I just wonder what is next."`
